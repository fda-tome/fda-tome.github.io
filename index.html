Currently

I'm currently working on my master's dissertation, developing a new method for large-scale wave simulation on modern accelerators. The approach adapts algorithms that were focused on matrix multiplications (e.g., COSMA) to matrix factorizations and orthogonalization methods such as QR and Schur, and functional analysis algorithms, including the Schur-Parlett recurrence for matrix functions. I mainly write research scripts in Julia using Dagger.jl (which I co-maintain), and my production code spans Julia, C++, and CUDA.

I'm also a Research Software Engineer at the MIT JuliaLab, where I build interpretable abstractions that make parallel and distributed computing and numerical linear algebra parameter tuning easier to write, faster to run, and performance-portable.
